# DSM-5-TR

## Descrição

Esta pasta é destinada aos materiais de referência relacionados ao
**Diagnostic and Statistical Manual of Mental Disorders – Fifth Edition – Text Revision (DSM-5-TR)**.

## Observação

O DSM-5-TR é protegido por direitos autorais da American Psychiatric Association (APA) e, por esse motivo, seu conteúdo não é distribuído neste repositório.

Durante o desenvolvimento do projeto será utilizada apenas uma referência autorizada ou um resumo permitido pela instituição responsável.

## Finalidade no Projeto

O DSM-5-TR será utilizado como fonte de apoio para:

- identificação de sinais e sintomas;
- contextualização das respostas do assistente;
- apoio à recuperação de informações no pipeline RAG.
