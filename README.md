# Zophia Lite RAG

## Sobre o Projeto

O Zophia Lite é um assistente inteligente voltado ao apoio educacional em saúde mental.

O sistema utiliza a arquitetura Retrieval-Augmented Generation (RAG), combinando modelos de linguagem com recuperação de informações em documentos científicos.

Este projeto está sendo desenvolvido para a disciplina de Inteligência Artificial da Universidade Estadual da Paraíba (UEPB).

---

## Objetivos

- Desenvolver um assistente conversacional.
- Utilizar documentos científicos como base de conhecimento.
- Gerar respostas fundamentadas.
- Não realizar diagnósticos médicos.
